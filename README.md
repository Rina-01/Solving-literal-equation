# Solving-literal-equation

Вводятся 3 выражения, являющиеся двумя слагаемыми и их суммой, а компьютер подбирает числовые варианты, в выбранной системе счисления (максимальная система счисления - 10).

Пример 1:

   удар
   удар
  драка
     10
     
Ответ:
   8126+8126=16252


Пример 2:

   папа
   мама
   дитя
     10
     
Ответ:
   Нет вариантов
